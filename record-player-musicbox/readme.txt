一个仿古风的唱片背景音乐播放器，支持自动播放、禁播设置、自定义位置、进度记忆及移动端控制等功能。

== Description ==

🎵 **唱片播放器 MusicBox** 是一个灵感来源于复古风播放器的 WordPress 插件，为你的网站添加可拖拽、可记忆播放位置的迷你唱片播放器。

**主要功能：**

- 支持自动播放和禁播时间设置
- 播放器位置自定义（left/bottom）
- 播放进度记忆，可手动开启/关闭
- 手机端禁播控制
- 首页专属显示选项
- 随机播放音乐列表

**特色亮点：**

- 真实的唱片旋转动画
- 双击播放器可切换“禁止自动播放”状态
- 本地缓存播放状态，避免多标签页重复播放
- 高度可配置，适配多种主题风格

== Installation ==

1. 上传插件到 `/wp-content/plugins/` 目录。
2. 在 WordPress 后台的“插件”页面启用插件。
3. 进入“设置 > 唱片播放器设置”页面，自定义音乐源和播放器参数。
4. 保存后，刷新前台页面查看效果。

== Frequently Asked Questions ==

= 如何添加多首音乐？ =
在设置页的“音乐源地址”文本框中，每行输入一个 mp3 链接，插件将自动随机播放。

= 播放器为什么在移动端不显示？ =
默认情况下，移动端禁用播放器自动播放。如需显示，请关闭“禁止手机端自动播放”选项。

= 为什么播放器不会自动播放了？ =
如果你双击过播放器图标，插件会在本地存储一个禁播时间（默认 48 小时）。你可以点击后台“清除禁播”按钮或刷新页面后双击播放器解除。
